Theme Review

Let's get set up!

You should have a test environment. If you do not have a local install of WordPress and are at a contribution day, let the lead or organiser know.

Import the theme unit test data

Next, it's time to import the theme unit test. This will allow you to really put that theme through it's paces. We have included the test data in this pack should you need it. You can get it right here.

Set WP_DEBUG to 'true' in wp-config.

Install helpful plugins

Install the following Plugins (Note: all plugins below can be installed via the Developer plugin):

  • Theme Check
  • Debug Bar
  • Log Deprecated Notices
  • Monster Widget
  • WordPress Beta Tester
  • Regenerate Thumbnails
